# BlogApp Backend

BlogApp Backend is build with  
1) nodeJS(express),  
2) joi validation,
3) Postgres DB
4) JWT AuthToken

## Installation

```bash
npm i
```

```bash
cp .env.example .env
```
complete `.env` with proper details
## Run

```bash
nodemon
```

## Postman Collection
You can find postman collection in `docs\blogappAPI.json`
